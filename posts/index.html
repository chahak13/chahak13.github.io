Gaussian quadrature in scipy

2021-07-14 :: [Chahak Mehta]
#python  #maths 
Scipy has an gaussian quadrature integration built-in in the integrate module as the integrate.quadrature function. import numpy as np from scipy import integrate A simple example of quadrature integration can be seen as follows, where we integrate the simple function \(f(x) = 2x\) from limits 0 to 2. f = lambda x: 2*x print("Quadrature integration:", integrate.quadrature(f, 0, 2)) print("Analytical solution:", 2**2) A more complex function can also be passed to the quadrature function.

Suppress warnings in python

2021-07-14 :: [Chahak Mehta]
#python 
While working with packages like numpy and scipy, python can possibly give a lot of warnings. These warnings can be suppressed easily by using the warnings module in the python library. import warnings warnings.filterwarnings("ignore") And done. This should suppress all the warnings your code produces. NOTE OF CAUTION: Do this only if you know what you’re doing and what warnings are being suppressed. Most of the times, having a warning means that something in your code can be improved.
